# LKNgm - Language of Knowledge (Gemini Edition)

Welcome to the central repository for the **LKN** ecosystem. This project focuses on the intersection of natural language, symbolic logic, and autonomous reasoning.

## Projects in this Repository

### 1. [lkngmSemantic](./lkngmSemantic)
A specialized reasoning engine that transforms Natural Language (NL) definitions into a graph-based knowledge representation for deductive inference.
*   **Current Focus**: Euclidean Geometry (Definitions 1-22, Book 1).
*   **Status**: Functional parity with the non-AI version has not been reached yet. Reaching 100% logical alignment requires manual refinement of the code generated by Gemini 3.
### 2. lknProgramming (Coming Soon)
The core LKN evaluator and rule engine. This project will host:
*   The Smalltalk-inspired programming language.
*   The Actor-based concurrency model.
*   The functional library used by the semantic engine for active rule execution.

## Collaboration
This repository hosts the collaborative development between the original creator of LKN and **Gemini**, focusing on modernizing the architecture for massive scale and advanced AI integration.
